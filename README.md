# xsetwall

a very tiny (120 LOC) X11 widgets utility

## screenshots

![macaron](./macaron.png)

## features

- set image as widgets
- customize position, width, height of widegts

## usage

```sh
macaron -x 50 -y 50 -w 250 -h 250 /path/to/img
```

## installation

1. clone the repo
2. run `sudo make clean install`
